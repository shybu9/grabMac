# grabMac
  `THIS TOOL IS USED TO FIND THE MAC ADDRESS OF AN IP
  
## USAGE 

### PRE-REQUIREMENTS 
* python3 :
```bash
    sudo apt-get install python3
```



* script_requirements :
```bash
    sudo pip3 install -r requirements.txt
``` 
### DOWNLOADING_SCRIPT :
```bash
    sudo git clone
```

### SYNTAX : 
```bash
   sudo python3 grabMAC.py -i <ip_address>
```

### EXAMPLES :
`$ sudo python3 grabMAC.py -i 127.0.0.1 `
